# TrabalhoPAM

Este é o repositório do projeto "TrabalhoPAM", desenvolvido como o último trabalho do ano para a <instituição escolar não revelarei> O projeto consiste em um jogo de TIC TAC TOE (Jogo da Velha) que faz uso de extensões e notificadores nativos para dispositivos Android.

## Sobre o Projeto

O objetivo deste projeto é criar um jogo interativo de TIC TAC TOE que demonstra o uso de tecnologias de Programação de Aplicativos Móveis (PAM) para Android. O jogo é uma versão moderna e interativa do clássico TIC TAC TOE, com recursos adicionais que utilizam extensões e notificadores nativos.

## Recursos

- Jogo de TIC TAC TOE interativo.
- Utilização de extensões para funcionalidades adicionais.
- Notificações nativas para melhorar a experiência do usuário.
